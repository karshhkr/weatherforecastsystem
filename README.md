#  Weather Forecast Web Application

A simple and responsive **Weather Forecast Web App** that allows users to search any city and get real-time weather information such as temperature, humidity, and weather conditions.

This project was developed during my **Web Development Internship at CodSoft**.

---

##  Features

*  Search weather by **city name**
*  Display **temperature and humidity**
*  Dynamic **weather icons**
*  **Responsive user interface**
*  Real-time weather data using **API**

---

##  Tech Stack

* **HTML**
* **CSS**
* **JavaScript**
* **REST API**
* **JSON**

## ⚙️ How It Works

1. User enters a **city name**.
2. The application sends a request to the **Weather API**.
3. The API returns weather data in **JSON format**.
4. The weather details are displayed dynamically on the webpage.
